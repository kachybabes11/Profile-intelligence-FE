Insighta Labs+ Web Portal

Overview
The Insighta Labs+ Web Portal is a browser-based interface that allows users to interact with the Profile Intelligence System.

Base URL: https://profile-intelligence-fe-production.up.railway.app/

It provides a user-friendly way to:
- Authenticate via GitHub OAuth
- View profile data

The frontend communicates with a centralized backend API.

 Features
- GitHub OAuth Authentication
- Secure Session Management (HTTP-only cookies)
- Role-Based Access Control (UI-level enforcement)
- Responsive User Interface

System Architecture

Browser (Frontend) ───► Backend API ───► Database

- Frontend handles UI and user interaction
- Backend handles logic and data processing
- Authentication is managed via secure cookies

Authentication Flow

1. User clicks "Login with GitHub"
2. Browser redirects to backend OAuth endpoint
3. Backend redirects to GitHub
4. GitHub redirects back to backend callback
5. Backend sets HTTP-only cookies
6. User is redirected to dashboard

Session Management

- Authentication is stored in HTTP-only cookies
- Protects against XSS attacks

CSRF Protection

- CSRF tokens are generated by the backend
- Tokens are included in requests that modify data
- Prevents unauthorized actions from external sites
